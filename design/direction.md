# Design Direction — Sprout

> *Neutral worked example. This is the durable design truth, extracted from the Claude
> Design project. The original Claude Design zip is unzipped into the code repo's
> `design-reference/` folder as first-build scaffolding and then discarded — this file is
> what every build reads.*

## Chosen direction: "Quiet Care"

Calm, forgiving, non-clinical. The opposite of a productivity app. The user already feels
guilty; the design must never add pressure.

## Reference anchors (why each)

- **Headspace** — for calm, rounded, non-judgmental tone.
- **Things (Cultured Code)** — for restraint and the satisfying "done" interaction.
- **A healthy plant in soft daylight** — the emotional target state.

## Tokens

- **Palette:** warm off-white background (#F7F5F0), soft sage green primary (#7C9A78),
  muted terracotta accent (#C97B5A) for the watered-confirmation moment only. No red, ever —
  no error-state alarm colours on a guilt-sensitive user.
- **Type:** one humanist sans (e.g. Inter), generous line height, large touch targets.
- **Corners:** fully rounded (16px). Nothing sharp.
- **Motion:** the "Watered" tap triggers a brief, soft bloom animation in terracotta — the
  single moment of delight. Everything else is still and quiet.

## Signature elements

- The plant card is the hero: large, photographic, calm.
- The watered-confirmation bloom is the one memorable interaction.
- Empty state is reassuring, never nagging: "All your plants are happy today."
